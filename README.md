# 🚀 Webro AI

Webro AI is an AI-powered website generation platform that transforms natural language prompts into modern, responsive websites within seconds. Simply describe the website you want, and Webro AI generates the structure, content, and layout using advanced AI models.

## ✨ Features

* 🤖 AI-powered website generation
* 🎨 Modern and responsive UI
* 🔐 Secure user authentication
* 📂 Project management dashboard
* ⚡ Real-time website generation
* 🌙 Clean and intuitive user experience
* 📱 Mobile-friendly design
* ☁️ Cloud deployment support
* 💳 Subscription and payment integration
* 🔄 Save and manage generated projects

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Shubham123-k/Webro-AI.git
cd Webro-AI
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm run server
```

### Environment Variables

Create a `.env` file inside the server directory:

```env
DATABASE_URL=YOUR_DATABASE_URL
BLUESMINDS_API_KEY=YOUR_API_KEY
BLUESMINDS_BASE_URL=YOUR_BASE_URL
BETTER_AUTH_URL=YOUR_URL
TRUSTED_ORIGINS=YOUR_FRONTEND_URL
```
